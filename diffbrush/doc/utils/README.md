# utils/ — 通用数学工具

**代码路径建议：** `utils/`

## 定位

放置静态方法或静态类。这些工具函数**按需创建**——当某个数学/几何运算被大量或所有同类型笔刷共用时，才提取到 `utils/` 下。不预先创建可能不必要的内容。

## 准则

- **静态方法或静态类**：不持有实例状态。
- **按需创建**：先有重复使用需求，再有工具提取。不凭空预构。
- **复用门槛**：如果一种运算被多个笔刷实现（或一个笔刷内的多处）用到，就提取到 `utils/`。如果仅一处使用，留在笔刷实现内部。
- **文件拆分**：按数学领域拆文件（例如 `bezier.py`、`stamp.py`、`compositing.py`），待实际需求出现时再建。
