# BezierUniformBrush — 单色均匀粗细贝塞尔笔刷

**代码路径建议：** `implementations/bezier_uniform_brush.py`

## 定位

一种单色、粗细均匀的三点 quadratic Bézier 笔刷。继承自 `BrushBase`。

## 与 BoaP 论文笔刷的差异

| 特征 | BoaP 论文笔刷 | 本笔刷 |
|---|---|---|
| 颜色 | 渐变色（双色 / 多色） | **单色** |
| 粗细 | 沿弧可变 | **头尾相同（均匀）** |
| 控制点 | 多点（cubic Bézier） | **三点：起点、控制点、终点** |

## 参数集

所有参数均为**归一化输入**（详见 [设计范式](../design_paradigm.md#参数归一化输入)）。几何相关以标准 patch 大小为基准归一化，颜色和透明度为 [0,1] 归一化。

```
Θ = { P₀, P₁, P₂, c, r, α }
```

| 参数 | 含义 | 形状 |
|---|---|---|
| `P₀` | 起点坐标 | `(2,)` |
| `P₁` | 控制点坐标 | `(2,)` |
| `P₂` | 终点坐标 | `(2,)` |
| `c` | 单色 RGB 颜色 | `(3,)` |
| `r` | 均匀半径（标量） | `(1,)` |
| `α` | 透明度 | `(1,)` |

## 数学形式

### 曲线

quadratic Bézier：

```
B(t) = (1-t)²·P₀ + 2(1-t)t·P₁ + t²·P₂,  t ∈ [0,1]
```

### 渲染

采用 Birth-of-Painting 3.2.1 的 **nearest-stamp parallel 模型**的 soft 化版本。
因本笔刷为单色（所有 stamp 颜色相同 c），"nearest-stamp 取色"退化为只需求"像素
是否落入笔刷覆盖区"——即所有 stamp 的覆盖并集。把论文的硬 disk（`||x−x_k|| ≤ r`）
与硬 SDF 零等值面替换为可微的 soft 形式：

**Soft disk（相对半径归一化，分辨率无关）：**

```
disk_k(x) = σ( γ · (1 − ||x − x_k|| / r) )
```

γ = `disk_softness`（无量纲边缘陡度）。`||x−x_k|| < r` 时 → 1，`> r` 时 → 0，
边界处 soft 过渡。对 stamp 中心 x_k（→Bézier 参数）和半径 r 可微。

**Stamp 激活权重（来自 Sigmoid 软计数）：**

```
w_k = σ( α_cnt · (L/d − k) )        # 即软计数 N = Σ_k w_k 的第 k 项
```

**覆盖并集（soft OR / soft-max over stamps）：**

```
m_k(x) = w_k · disk_k(x)
M(x)   = 1 − ∏_{k=1}^{K} ( 1 − m_k(x) )      ∈ [0,1]
```

概率并集 `1 − ∏(1−·)` 是 "像素落入任一 stamp" 的可微 soft 版本（等价于对
`||x−x_k|| − r` 取 soft-min 的 SDF 零等值面 soft 化）。乘积形式数值稳定，无需 log
空间或钳位。

**Alpha compositing 输出（预乘 RGBA）：**

```
A(x) = α · M(x)                      # 像素不透明度
RGB(x) = c · A(x)                    # 预乘颜色
patch = concat(RGB, A)  ∈ (B, 4, H, W)
```

输出预乘 RGBA，便于下游 "over" 合成：`canvas' = patch + canvas·(1−A)`。
梯度通路：`patch → A → M → {disk→x_k→t_k→s_k→d→r, disk→x_k→Bézier, w_k→L/d→{L→Bézier, d→r}}`，
全部参数（P₀,P₁,P₂,c,r,α）均收到有限梯度。

**坐标映射：** 归一化坐标 (x,y)∈[0,1] → 像素 (x·W, y·H)；半径 r → r·min(H,W)。
像素中心位于 (j+0.5, i+0.5)。

## Stamp 采样策略

### 与 BoaP 的差异

| | BoaP 论文 | 本笔刷 |
|---|---|---|
| 采样方式 | 固定 N，在曲线上均匀取 t | **基于间隔距离的比例采样** |
| N 的决定 | 固定超参数 | **由曲线长度和比例参数动态计算** |
| 效果 | N 固定，短曲线 stamp 过密，长曲线 stamp 过疏 | **任意形状曲线均保持均匀一致的 stamp 密度** |

### 采样逻辑

引入一个比例参数 ρ（或归一化等价形式），stamp 的放置间隔由笔刷半径 r 和 ρ 共同决定：

```
间隔距离 d = r × ρ
```

stamp 沿曲线的放置规则：**曲线每前进距离 d，放置一个 stamp**。

等价于：

```
总曲线长度 L = ∫₀¹ ||B'(t)|| dt

N = f(L, d)    ← N 的计算方式待定（见下方"可微性说明"）
```

### 目的

让 stamp 在任何形状的曲线（长 / 短 / 弯 / 直）上保持均匀且一致的覆盖密度，避免固定 N 采样导致短曲线 stamp 堆积或长曲线 stamp 稀疏的问题。

### 可微性说明

**方案：Sigmoid 软计数。**

```
N = Σ_{k=1}^{K} σ(α · (L/d - k))
```

其中 σ(x) = 1/(1+e⁻ˣ) 为标准 sigmoid 函数，α 为陡度参数。

**余段正常化。** 此方案与 floor 一样，在曲线尾部会产生一段不足 d 的"多余尾巴"（余段）。这是采样逻辑的结构性正确行为——最后一段不够一个 d 就不该再盖一个 stamp。**这不是问题，不需处理。** 余段的存在同时为梯度提供了"L 正在接近下一个整数"的信号，优于 floor 的完全截断。

**实现时注意。** α 和 K（最大 stamp 数上界）作为 `self` 变量写在类的 `__init__` 中（硬编码赋值），不作为外部常量或全局变量。
